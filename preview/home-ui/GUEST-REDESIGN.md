# 비회원 메인 리디자인 (대치동 데모)

**기준:** [9장 SSOT](../../docs/ssot/09-main-screen-roles.md) §4 · 메모(메인화면·광고·등급·반응형)  
**프리뷰:** http://localhost:5174/#/guest

## 변경 포인트

| 영역 | 변경 내용 |
|------|-----------|
| **지역 컨텍스트** | 샘플 지역 **서울 강남구 대치동** 고정 · 동 우선 정책 · 공부방/과외/학생 의뢰 요약 수치 |
| **지도** | 대치동 중심 플레이스홀더 · 존재감 중간(히어로 우측·모바일 하단) · 공부방 전용 |
| **공부방** | Prime 3 + Pick 5×2행 박스형 · 오렌지 악센트 · 찜/비교/문의 로그인 유도 |
| **과외쌤** | 동일 3+5×2 구조 · 블루 악센트 · 지도 없음 |
| **학생** | 카드형 제거 → **리스트형** · 강남·서초·송파 권역 의뢰 더미로 풍성하게 |
| **광고** | 점선 박스 → **프로모 카드**(프리미엄/제휴/안내) · 데스크톱 사이드바 · 모바일 인라인 |
| **로그인 유도** | 상세·찜·비교·문의·지역변경 → confirm 후 auth-ui 로그인 · 하단 스트립 + 헤더 버튼 |
| **시각 리듬** | 박스형(공부방/과외) vs 리스트형(학생) 대비 · 올드 포털형 CTA 배너 제거 |

## 반응형

- **데스크톱(≥1024px):** 본문 + 우측 프로모 사이드바 · 히어로 2열(지역+지도)
- **태블릿:** Prime/Pick 그리드 3·5열 축소
- **모바일:** 히어로 세로 스택 · 사이드바 → 본문 상단 인라인 프로모 1장 · 학생 리스트 세로 정렬

## 로그인 후 원칙 반영 (염두)

- 이번 화면: 공부방·과외 **광고/프리미엄만 박스형**으로 시연
- 로그인 역할 화면(`#/parent` 등)은 기존 3/5/리스트 유지 — 일반 항목 리스트화는 추후

## [임시] 처리 항목

| 항목 | 비고 |
|------|------|
| 지도 API | CSS 플레이스홀더 · 대치동 좌표 미연동 |
| 지역·집계 수치 | `GUEST_REGION_STATS` 더미 |
| Prime/Pick **상품 노출명** | SSOT `[임시]` |
| 광고 슬롯 구매·외부 배너 | 프로모 카드 + alert |
| GNB·더보기 라우팅 | alert / login-gate |
| auth-ui 연동 | localhost:5173 링크 |
| 실데이터 API | 미구현 |
| 로그인 gate | confirm → 새 탭 (모달/토스트 추후) |

## 확인 방법

```bash
cd preview/home-ui && npm run dev
```

1. `#/guest` — 비회원 메인  
2. 창 너비 조절 또는 DevTools — 모바일·데스크톱  
3. 슬롯 카드·학생 행·찜 버튼 클릭 → 로그인 유도
