## 3. Cursor 병행 작업안

### 3.1 권장 병행 패턴

```
[Track A] UI 프리뷰 (Vite 정적)     ←── SSOT 화면·필드 잠금 준수
[Track B] docs + SQL                ←── SSOT ↔ DDL 동기화
[Track C] PHP MVC                   ←── 프리뷰 확정 후 이식
```
