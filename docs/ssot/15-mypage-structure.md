## 1. 목적

마이페이지는 **설정 화면이 아니라 개인 운영 허브**다.  
15장은 **입력 필드 장(14장)이 아니라 운영 레이어 장**이다 — 등록 폼 정의·상품 카탈로그·쪽지 UI 본문은 각각 14·18·16장에 둔다.

| 축 | 내용 |
|----|------|
| 탐색 후속 | 찜 · 최근열람 · (학부모 중심) 비교 후속 |
| 공급 운영 | 내 등록 · 공개/숨김 · 상세등록 CTA |
| 접촉 | 쪽지함 진입(16장) |
| 수익화 | 유료 요약·진입(18장) — **공급자 중심** |
| 계정 | 역할 · 지역 · 수신동의 |

### 1-1. 잠금 원칙

- **허브:** 찜 / 최근열람 / 유료 / 쪽지 / 등록 상태가 **한 화면(홈)에서 연결**되어야 한다.
- **역할:** 페이지 **틀은 동일**, **강조 순서·첫 탭·홈 CTA만** 역할별로 다르다(6·9장).
- **다음 행동:** 사용자는 항상 「지금 무엇을 해야 하는지」를 본다. **홈 숫자(찜 N, 쪽지 N, 공개중 N)는 장식이 아니라 다음 CTA의 입력값**이다.
- **학부모:** 자녀 0명이면 홈 장식보다 **자녀 등록 CTA**를 우선한다.
- **학생 의뢰:** 민감정보·공개 범위는 19장과 연동 · 마이페이지 숫자와 일치.
- **시스템 안내 vs 쪽지:** 상품 만료 · 공개 준비 미완료 · 제출자료 상태 등 **시스템 안내**와 **회원 간 쪽지**는 개념을 분리한다(1차: 쪽지=16장 · 시스템 안내=후순위). **「검증 반려」「운영 알림」 등 운영자 심사·반려 느낌의 용어는 쓰지 않는다** — [22장 §7](22-platform-lifecycle-principles.md#7-용어-정리-1차).

**비회원:** 마이페이지 전부 접근 불가 → 로그인 유도(9장 §13).

### 2-1. P15-10 제출자료 상태

**운영자 심사·승인·반려 화면이 아니다.** 사용자가 학력·경력·자격 등 **관련 자료를 등록**하고 **공개 여부를 선택**한 뒤, 플랫폼은 **제출 여부·공개 상태만 표시**한다. 사실 여부를 보증하거나 반려하지 않는다. 학부모는 공개된 자료·프로필을 바탕으로 **직접 판단**한다.

| ✕ 금지 (UI·문구) | ○ 허용 (상태 표시) |
|------------------|-------------------|
| 승인 · 반려 · 검수중 · 심사 대기 | 미제출 · 제출함 · 수정 가능 |
| 운영자 보완 요청 | 공개함 · 비공개 · 선택 제출 |
| 「검증 통과/실패」 | 본인 확인용 · 학부모 공개용 (범위 라벨) |

- **1차 주체:** `tutor` (과외쌤) — [21장](21-tutor-registration-management.md)
- **공부방:** 동일 원칙 · 증빙 테이블은 [5장](05-study-room-db.md) · 1차 UI는 후순위(권한표 △)
- **DB 테이블명** `*_verification_documents`는 레거시 명칭 — **운영자 검수 의미로 해석하지 않음** ([22장 §5](22-platform-lifecycle-principles.md#5-본-원칙과-다른-축-혼동-방지))

## 부록 A. 구현 추적

> **1차 프리뷰 정본:** hash `#/mypage/*` · 역할별 default path는 `router.js` `getDefaultMypagePath`.

| 후보 | 화면 | 메모 |
|------|------|------|
| `/mypage` | P15-01 | PHP MVC 후보 |
| `/mypage/wishlist` | P15-06 | |
| `/mypage/registrations/...` | P15-02~05 | |
| `#/mypage/*` (hash) | P15-xx | **home-ui 프리뷰** — `http://127.0.0.1:5174/#/mypage/home` 등 |

> **프리뷰 주의:** home-ui는 hash 라우팅이다. 부록의 `/mypage/...` 경로만 붙이면 `#/guest`로 떨어질 수 있음 → `#/` 접두 또는 툴바 「마이페이지」 사용.

프리뷰 분리(auth / home / study-room-ui / tutor-ui)와 최종 PHP 단일 라우팅 **아직 한 방향 미잠금**.
