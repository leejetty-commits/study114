## 1. 등록 2단계 (잠금)

| 단계 | 내용 | 노출 |
|------|------|------|
| **기본등록** | 최소 항목 빠른 입력 | **일반 리스트** 노출 가능 |
| **상세등록** | 소개·과목·가격·이미지·활동지역·증빙 등 | 누구나 **이어서** 가능 |

---

## 2. 등급(Prime / Pick)과의 관계

| 규칙 | 내용 |
|------|------|
| Prime / Pick 박스 노출 | **상세등록 완료 필수** |
| 상세등록 접근 | **등급 구매자만** 제한하지 않음 — 누구나 가능 |
| 미완료 시 | 박스형 상위 노출 **불가** · 일반 리스트는 가능 |

---

## 3. 유튜브 링크 (1차)

| 항목 | 규칙 |
|------|------|
| 허용 | **외부 YouTube URL 1개** |
| 업로드 | 직접 업로드 **아님** |
| 배치 | **상세등록** 영역만 |
| UI | 썸네일 + 버튼 수준 (1차) |
| 제외 | 영상 과금 · 열람권 결제 · 내부 플레이어 복잡화 |

---

## 4. 학생 열람권 (후보 · 1차 미구현)

| 항목 | 규칙 |
|------|------|
| 범위 | **과외쌤 전용** 수익모델 후보 |
| 공부방 | 핵심 구조로 **두지 않음** |
| 형태 | 단순 기간제 X → **건수 + 유효기간** 혼합 검토 |
| 예시 | 10건 / 30일 · 20건 / 60일 |
| 1차 | 설계·메모만 · **결제 구현 제외** |

## 5. 노출·비교 필드 (11장 연동 · 설계)

| UI | 테이블·컬럼 |
|----|-------------|
| 표시명 | `tutors.display_name` |
| 프로필 | `tutor_images.image_path` |
| 주요 과목 | `tutor_subject_targets.subject_name` |
| 가능 지역 | `tutor_regions` |
| 대표 과외비 | `tutors.preferred_fee_amount` |
| 경력 | `tutors.career_years` |
| 소개/특징 | `intro_short`, `feature_1`~`3` |
| 비교(경량) | `gender`, `lesson_format`, `education_background_note`, `verification_available` |
