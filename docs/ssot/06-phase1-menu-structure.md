## 2. 안전과외 (§3)

| 구분 | 규칙 |
|------|------|
| 정의 | 알고리즘·에스크로·지급 중개 **아님** — 선입금 주의·분쟁 예방 **설명형 가이드** |
| 1차 위치 | **고객센터 / 이용안내** 하위 설명 페이지 |
| 1차 범위 밖 | 결제 보류 · 지급 자동화 · 실제 에스크로 |

---

## 3. 공부방 비교검색 (§4 · 1차 필수)

| 규칙 | 내용 |
|------|------|
| 최대 건수 | **3개** 공부방 |
| 선택 | 사용자가 직접 선택 |
| UI | **로그인 후 팝업(모달) 표 형태** |
| 비회원 | **열리지 않음** |
| 자격 | 등급보다 **비교 필수 항목 충족** 우선 |
| 등급 | 핵심 비교 항목은 등급별로 바꾸지 않음 · Prime/Pick은 보조 정보만 추가 |

기본 비교 항목 후보: 공부방명 · 위치(동/단지) · 대상 학년 · 주력 과목 · 가격 대표값 · 수업 형태 · 1타임 인원 · 주말 가능 · 교육청 등록 · 시설 핵심 · 특징 1~3

---

## 4. 찜 기능 (§5 · 1차 필수)

| 규칙 | 내용 |
|------|------|
| 대상 | **공부방 + 과외쌤** 모두 |
| 1차 범위 | 로그인 사용자 찜 · 마이페이지 목록 · 찜 해제 |
| 연동 | 찜한 공부방 → 공부방 비교 후보 · 찜한 과외쌤 → 과외쌤 비교 후보 |

---

## 5. 1차 포함·원칙

| 항목 | 규칙 |
|------|------|
| 공부방 비교검색 | **필수** (§3) |
| 과외쌤 비교검색 | **경량** · 공부방보다 항목 적게 |
| 학생 비교검색 | **두지 않음** |
| 가입 회원구분 표기 | **「학생/학부모」** 유지 |
| 서비스 내부 메뉴·카피·탐색 | **「학생」** 중심 통일 |
| 상단 네비 | **유틸 메뉴 + 메인 GNB 2층** |
| 쪽지함 | 학생·학부모 ↔ 공부방·과외쌤 **핵심 중계 채널** (단순 알림 아님) |
| 공식 접촉 | **쪽지** 중심 · 전화·메일은 플랫폼 내부 기능 아님 |

---

## 6. 유틸 메뉴 (예시)

### 비회원

- 지역선택
- 이용안내 *(안전과외 가이드 포함)*
- 고객센터
- 로그인
- 회원가입

### 로그인 후

- 지역선택
- 쪽지함
- 최근열람
- 마이페이지
- 로그아웃

---

## 7. 메인 GNB (예시)

- 공부방찾기
- 과외쌤찾기
- 학생/학부모
- 공부방등록
- 과외쌤등록
- 고객센터

**GNB에 넣지 않음:** 퀵매칭 · **안전과외 독립 메뉴** · 앱다운로드 · 자료실/포털형 확장

**포함 방향(§6-1):** 공부방찾기 · 과외쌤찾기 · 학생/학부모 · **마이페이지(유틸)** · 고객센터
