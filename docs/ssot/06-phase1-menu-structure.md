---

## 2. 1차 포함·원칙

| 항목 | 규칙 |
|------|------|
| 공부방 비교검색 | **필수 유지** |
| 과외쌤 비교검색 | 가능 · **공부방보다 경량** |
| 가입 회원구분 표기 | **「학생/학부모」** 유지 |
| 서비스 내부 메뉴·카피·탐색 | **「학생」** 중심 통일 |
| 상단 네비 | **유틸 메뉴 + 메인 GNB 2층** |

---

## 3. 유틸 메뉴 (예시)

### 비회원

- 지역선택
- 이용안내
- 고객센터
- 로그인
- 회원가입

### 로그인 후

- 지역선택
- 쪽지함
- 최근열람
- 마이페이지
- 로그아웃

---

## 4. 메인 GNB (예시)

- 공부방찾기
- 과외쌤찾기
- 학생/학부모
- 공부방등록
- 과외쌤등록
- 안전과외
- 고객센터

**GNB에 넣지 않음:** 퀵매칭 · 앱다운로드 · 자료실/포털형 확장

---

## 5. 과외쌤 비교검색 (경량 · 1차)

최소 항목 예:

- 이름/표시명
- 성별
- 주요 과목
- 가능 지역
- 대표 과외비
- 경력
- 특징
